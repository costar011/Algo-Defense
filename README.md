## 📖 프로젝트 소개

**Algo-Defense**는 개발 입문자들이 프로그래밍 언어의 예약어(Keyword)와 CS 용어를 자연스럽게 익힐 수 있도록 설계된 **기능성 타이핑 디펜스 게임**입니다.

기존의 단순한 타자 연습과 달리, 사용자가 **자신이 학습하고 싶은 언어(Python, Java, HTML 등)를 선택**할 수 있으며, 게임 플레이 중 용어의 정의(Definition)를 시각적으로 노출하여 **학습 효과를 극대화**했습니다.

## ✨ 주요 기능

### 게임플레이 기능

- 🎯 **타이핑 디펜스**: 적을 타이핑하여 제거하는 액션 게임
- 📊 **점수 시스템**: 정확한 타이핑으로 점수 획득
- ❤️ **생명력 시스템**: 실수를 하면 생명력 감소
- 📈 **난이도 증가**: 점수가 높아질수록 게임 난이도 상승

### 학습 기능

- 📚 **언어별 용어 학습**: 8가지 프로그래밍 언어 지원
- 🔍 **용어 설명 제공**: 각 용어의 정의를 게임 중 확인 가능
- 📖 **용어 공부 모드**: 게임 전 미리 용어를 학습할 수 있는 전용 모드
- 🔄 **확장 가능한 데이터**: JSON 파일을 통해 새로운 언어나 용어 추가 가능

### 사용자 경험

- 🎨 **직관적인 UI**: 깔끔하고 명확한 인터페이스
- ⌨️ **실시간 입력 표시**: 타이핑한 내용을 실시간으로 확인
- 🎬 **카운트다운**: 게임 시작 전 준비 시간 제공
- 📱 **반응형 디자인**: 1200x800 해상도 최적화

## 🎮 프로그램 사용 방법

### 메인 메뉴

- **언어 선택**: 학습하고 싶은 프로그래밍 언어를 클릭하여 선택합니다.
  - 지원 언어: Python, Java, HTML, CSS, C++, JavaScript, C#, TypeScript
- **용어 공부**: 게임을 시작하기 전에 해당 언어의 용어를 미리 학습할 수 있습니다.

### 게임 플레이

1. **게임 시작**: 언어를 선택하면 3초 카운트다운 후 게임이 시작됩니다.

2. **게임 목표**:

   - 화면 위에서 내려오는 적(용어)을 정확히 타이핑하여 제거하세요.
   - 적이 화면 하단에 도달하면 생명력이 감소합니다.
   - 생명력이 0이 되면 게임 오버입니다.

3. **조작 방법**:

   - **타이핑**: 키보드로 용어를 입력합니다.
   - **Enter**: 입력한 단어를 확인하고 총알을 발사합니다.
   - **Backspace**: 입력한 글자를 삭제합니다.

4. **게임 요소**:
   - **점수**: 적을 맞추면 10점씩 획득합니다.
   - **생명력**: 화면 우측 상단에 하트(♥) 3개로 표시됩니다.
   - **난이도**: 점수가 높아질수록 적의 이동 속도가 빨라집니다.

### 용어 공부 모드

1. 메인 메뉴에서 **"용어 공부"** 버튼을 클릭합니다.
2. 학습하고 싶은 언어를 선택합니다.
3. 해당 언어의 모든 용어와 설명을 확인할 수 있습니다.
4. **마우스 휠**: 용어 목록을 스크롤합니다.
5. **ESC 또는 ENTER**: 뒤로가기

### 게임 오버

- 생명력이 모두 소진되면 게임 오버 화면이 표시됩니다.
- **ENTER 또는 SPACE**: 메인 메뉴로 돌아갑니다.
